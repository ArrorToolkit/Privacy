# Develop Accelerator

## 特性

1、消息推送

2、图标生成

3、Hysterai代理

## 使用

### APNs
1、添加证书文件。

2、输入组织、钥匙、套装、令牌等信息。

3、修改负载。

4、点击发送按钮。

### Icon Generator

1、添加图标。

2、点击导出按钮

### Hysterai代理

1、填写代理信息

2、点击Start按钮
